# test-backend-service

## Development

### prepare service with docker

```sh
docker-compose -d up
```

### install dependencies

```sh
yarn
```

### start dev service

```sh
yarn dev
```

### Folder structures

```sh
./src
├── api
├── config
├── lib
├── middleware
├── services
├── app.js
├── index.js
├── server.js
└── task.js
```
