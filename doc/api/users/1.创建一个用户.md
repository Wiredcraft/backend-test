# 1. 创建用户

**简要概述**

- 创建一个用户

**请求URL**

```none
/api/users/create
```

**请求方式**

- POST

**参数**

|   参数名    | 必选 |  类型  |      说明      |
| :---------: | :--: | :----: | :------------: |
|    name     |  是  | string |    用户名称    |
|     dob     |  否  | string | 用户的出生年月 |
|     age     |  否  | number |      年龄      |
|   address   |  否  | string |      地址      |
| description |  否  | string |    其他信息    |
|  password   |  是  | string |    用户密码    |
