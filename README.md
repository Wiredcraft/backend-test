## DEV DEPENDENCIES
1. docker@20.x
2. Yarn@1.22.x
3. Nodejs@16.x
## RUN IN DEPLOYMENT
```
$ yarn
$ yarn start:dev
```

## FOR E2E TEST
```
$ yarn e2e:test
```
## BUILD TO PRODUCT 
```
$ yarn
$ yarn build
```

## FOR SWAGGER DOCUMENT
Start server first and browser http://localhost:3000/api
