# A General User Management Service

## Installation

```bash
npm install
```

## Running the app

```bash
# development
npm run start

# watch mode
npm run start:dev
```

```
# build
npm run build

# production mode
npm run start:prod
```

## Test

```bash
# tests
npm run test

# test coverage
npm run test:cov
```
